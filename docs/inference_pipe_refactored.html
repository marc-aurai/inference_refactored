inference_pipe_refactored module

This is the main script of the Inference container on Azure. The container reads the preprocessed messages inside the inference queue, and sends a post request to the Azure ML studio endpoint. The endpoint will answer with a prediction, uuid, probabilities, prediction_list and timestamp -> created_at.
